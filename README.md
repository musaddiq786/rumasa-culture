# 👗 RUMASA CUTURE — Bridal & Party Wear Rental Website

> **Wear It. Flaunt It. Return It.**  
> A fully responsive rental e-commerce website for premium bridal and party wear.

🌐 **Live Demo:** [rumasa-culture.netlify.app](https://rumasa-culture.netlify.app)

---

## 📌 About the Project

RUMASA CUTURE is a single-page rental platform for women's ethnic and party wear. Built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies — just clean, fast, production-ready frontend code.

The site allows users to browse a curated collection, filter by category, check pricing, submit booking requests, and contact via WhatsApp — all in one seamless experience.

---

## ✨ Features

- 🛍️ **Product Collection** — Filterable grid with search, category filter & price sort
- 📋 **Booking Form** — Integrated with Formspree for real email delivery
- 💰 **Live Price Calculator** — Auto-calculates rent + deposit based on outfit and rental days
- 💬 **Quick View Modal** — View outfit details without leaving the page
- ❤️ **Wishlist** — Save favourite outfits (stored in localStorage)
- 📐 **Size Guide Modal** — Women's sizing chart built in
- ✅ **Availability Checker** — Demo booking conflict detection
- 📱 **WhatsApp Integration** — Floating button for direct customer contact
- 🔍 **SEO Ready** — Open Graph tags, JSON-LD schema markup, meta descriptions
- 📱 **Fully Responsive** — Works on mobile, tablet, and desktop
- ♿ **Accessible** — ARIA labels, keyboard navigation, semantic HTML
